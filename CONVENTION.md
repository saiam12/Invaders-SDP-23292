## 1. Branch Naming
    type/explain...
        develop
        feature
        hotfix
    ex) feature/add-shield-item

## 2. Commit Message
    type: title
        feat:
        fix:
        refactor:
        etc...
    ex) feat: add invincibility item logic
        refactor: simplify item drop controller

    if necessary, add body and footer

## 3. Pull Request Template
    I created a PR template in the master branch. If you want to use another template, you can create the PULL_REQUEST_TEMPLATE directory and put the template file inside.
